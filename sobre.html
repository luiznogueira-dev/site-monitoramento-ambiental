Sobre o Projeto

Este projeto foi desenvolvido como parte de um módulo educacional com foco na construção de aplicações web aplicando boas práticas de desenvolvimento. Ele simula uma aplicação de monitoramento ambiental com foco na qualidade do ar, da água e do solo.

Utilizamos HTML semântico, CSS responsivo e técnicas de acessibilidade para garantir uma boa experiência em diferentes dispositivos.
